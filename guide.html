使い方・画面の見方

このツールは、USD/JPY のサポート（支持）・レジスタンス（抵抗）ラインを、複数の時間足から自動で検出して表示します。「自分のライン分析を補助する」ことを目的にしたツールです。まずは何を表示しているのか、そしてどう使うのかを順番に説明します。

はじめに（重要）：本ツールは売買を指示・推奨するものではありません。「ここで買う／売る」を教えるサインツールではなく、あなた自身が引いたラインの信頼度を高めたり、見落としていた節目に気づいたりするための“補助”です。詳しくは下の「補助として使うという考え方」をお読みください。

1. このツールが計算していること

過去の値動きから「スイング高値・安値（一時的な天井・底）」を抽出し、価格が近い水準ごとにまとめて、よく反発している価格帯を S/Rラインとして表示します。各時間足（1分〜日足）ごとに独立して計算し、さらに「複数の時間足で重なっている価格帯」を合流（コンフルエンス）として強調します。

2. 画面の用語

表示意味
現在値USD/JPY の最新価格（5分ごとに更新）。
R1 / R2現在値より上にあるレジスタンス（抵抗）。価格が上で跳ね返されやすい目安。R1が直近、R2がその上。
S1 / S2現在値より下にあるサポート（支持）。価格が下で支えられやすい目安。S1が直近、S2がその下。
★（1〜3）そのラインの強度。直近でよく触れている・タッチ回数が多いほど★が増えます（★★★が最強）。
合流 / 合NそのラインがN個の時間足で重なっていること。数が多いほど多くの時間軸が意識する強い節目。
pips現在値からそのラインまでの距離（USD/JPYは0.01円＝1pip）。
接触 / 接触S・接触R今まさに価格がそのライン付近にある状態。S＝サポートに接触、R＝レジスタンスに接触。

3. 各パネルの見方

🔗 マルチTF合流ライン（強い順）

複数の時間足で重なっている価格帯を、重なりの多い順に並べています。「上の抵抗」「下の支持」と現在値からの距離（pips）も表示。まずここを見れば、いま相場全体が意識している強い価格帯が分かります。

📊 時間足別 S/Rライン

「全体」タブでは、1分足〜日足までの S2 / S1 ／現在値／ R1 / R2 を一覧表で比較できます。どの時間足のラインがどこに集まっているかが一目で分かります。各時間足タブ（1m〜1d）を押すと、その時間足だけのラダー表示に切り替わります。

4. S/Rライントレードの基本

S/Rライン（サポート・レジスタンス）は、ライントレードの最も基本的な考え方です。ざっくり言うと、

  • サポート（S）＝下値で買い支えられやすい価格帯。反発（上昇）を狙う・下抜けに警戒する目安。
  • レジスタンス（R）＝上値で売られやすい価格帯。反落（下落）を狙う・上抜け（ブレイク）に警戒する目安。
  • 多くの時間足が重なる節目（合流）ほど、反応が大きくなりやすいと言われます。

これらは「必ずそこで反転する」というものではなく、あくまで意識されやすい価格帯です。抜けることも当然あります。

5. 「補助」として使うという考え方（最重要）

このツールの狙いは2つだけです。
  1. あなた自身が引いたラインの“信頼度を上げる” — 自分が引いた水準と、ツールが自動検出した水準・★・合流が一致すれば、その根拠はより確かになります。
  2. あなたが“見落としていた”ラインに気づく — 自分が見ていなかった時間足の節目や、意識すべき合流ゾーンを拾い、見落としを減らします。

つまり、主役はあくまであなた自身の相場分析です。本ツールはその答え合わせと抜け漏れチェックをする“セカンドオピニオン”にすぎません。

こう使ってください

  • まず自分でチャートにラインを引く → ツールの結果と照らし合わせて一致／不一致を確認する。
  • 一致するライン＝根拠が増える。自分が引いていない強い合流ライン＝見落としかもしれないと点検する。
  • エントリー・利確・損切りの最終判断は必ず自分で。ツールの数字をそのまま注文にしない。

こう使わないでください

  • ★や合流が多いラインを「ここで必ず反発する」と鵜呑みにして機械的に売買する。
  • 自分の分析を一切せず、ツールの表示だけでエントリーする。
  • 本ツールを「儲かるサイン／自動売買シグナル」として扱う。そういうものではありません。

6. 更新ペースについて

画面は約5分ごとに自動で再読み込みされます。現在値と短い時間足は約5分ごとに最新化されます（手動で「🔄更新」を押すこともできます）。

ただし、データの取得（API更新）の間隔は時間足ごとに異なります。長い時間足は値動きがゆっくりなので、無駄な取得を避けるために更新頻度を落としています（その分、無料データの範囲で長時間の運用を可能にしています）。実際の更新間隔の目安は次のとおりです。

時間足データ更新の目安
1分足 / 5分足約5分ごと
15分足約15分ごと
1時間足約1時間ごと
4時間足約4時間ごと
日足約6時間ごと
  • 長い時間足のラインは、上記の間隔で更新されるまで前回の値のままになります（値動きが遅いため実用上の影響はほとんどありません）。
  • リアルタイム（ティック単位）の配信ではありません。表示価格・ラインは数分前のものになることがあります。
  • サーバー混雑などで、まれに更新が数分遅れる・スキップされる場合があります。
  • スキャルピングなど秒単位の判断には向きません。実際の発注時は、必ずご利用のFX会社のリアルタイムレートをご確認ください。
